## Common Questions

**Q: Do I need to move to a custom server?**
A: Probably not. Your current GitHub Pages setup is working great. Only move if you need specific features like custom email handling or SMS notifications.

**Q: What access do you need?**
A: SSH credentials, server IP, and admin access. See [CAN_YOU_HELP_WITH_SERVER.md](CAN_YOU_HELP_WITH_SERVER.md) for details.

**Q: How long does deployment take?**
A: 30-60 minutes for a basic custom server setup following the checklist.

**Q: Is my contact information secure in these docs?**
A: Yes. The emails and phone numbers in these docs are already public on your website's contact page. This is your business contact information meant to be public.

**Q: Which deployment option should I choose?**
A: See comparison in [SERVER_MAINTENANCE_GUIDE.md](SERVER_MAINTENANCE_GUIDE.md). GitHub Pages is recommended for most cases.
